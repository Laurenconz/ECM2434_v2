Description

This is a BINGO game designed to promote sustainability on campus, where players complete tasks to earn points and rewards.

There are nine games. When a user completes a task, it turns into a stamp, and they earn points. Each game is worth a set number of points and requires either visiting a location or uploading a picture. For location-based tasks, players must scan a QR code to verify they have been there. Players can earn bonus points for completing tasks in a specific pattern.

The game has a time limit of one month, with bonus points for early completion. The top three players with the highest scores will receive rewards such as gift cards.

API Documentation

This section provides detailed guidelines explaining how developers can use the API (Application Programming Interface)

Authentication

This section explains how to authenticate API requests. It describes how users or developers must provide proof of identity when making API calls.

Locations API

- GET /api/locations - List all game locations
- POST /api/locations - Create a new location
- GET /api/locations/{id} - Get location details
- PUT /api/locations/{id} - Update a location
- DELETE /api/locations/{id} - Remove a location

Photo Verification API

- POST /api/photos - Submit a photo for verification
- GET /api/photos/pending - List pending verification photos
- PUT /api/photos/{id}/verify - Mark a photo as verified

Photo Verification System

This system allows users to submit photos from locations, which can then be verified by game keepers.

How It Works

1. Users take photos at specific locations
2. Photos are uploaded to the game board
3. Game keepers review photos through the admin panel
4. Verified photos allow users to gain points on the leaderboard

QR Code System

This system allows users to scan QR codes at specific locations.

How It Works

1. If a game on the game board requires the user to scan a QR code, a scanner will appear
2. The scanned QR code will be checked against our data to verify its correctness
3. If the QR code is valid, users will gain points on the leaderboard
